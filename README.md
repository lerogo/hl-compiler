## 词法分析器 ##
### 介绍 ###
需要两个文件
 - 解析的代码文件
 - 配置文件 见`src/com/lerogo/compilar/lexer/config.json` json配置
   - 关键字 KEY_WORD
   - 运算符 OP
   - 界符 SYMBOL
   - 标识符 ID （正则表达式）
   - 常量 CONSTANT （正则表达式）
